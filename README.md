# 小学生算数アプリ (sannsuuApp)

C# と WPF (Windows Presentation Foundation) で作成した、小学校1年生から6年生まで対応の算数クイズアプリです。

## 🚀 特徴

- **学年別出題ロジック**: 1年生の「10までの足し算」から6年生の「文字式(x)」まで、学年に合わせた問題を自動生成。
- **直感的な操作**: 学年を選んで「スタート」を押すだけで、すぐに学習を開始できます。
- **リアルタイム判定**: 答えを入力してボタンを押すと、その場で正誤を判定。

## 🛠 開発環境

- **言語**: C#
- **フレームワーク**: .NET 6.0 / 8.0 (WPF)
- **開発ツール**: Visual Studio 2022

## 📝 実装済み機能

| 学年 | 内容 |
| :--- | :--- |
| 1年生 | 10までの足し算 (合計が10以下) |
| 2年生 | 九九 (かけざん) |
| 3年生 | 3桁の引き算 |
| 4年生 | 割り算 (あまりなし) |
| 5年生 | 小数の足し算 (小数点第1位) |
| 6年生 | 文字 x を使った式の逆算 |

## ⚙️ 使い方

1. リポジトリをクローンまたはダウンロードします。
2. `sannsuuApp.sln` を Visual Studio で開きます。
3. `F5` キーを押して実行します。
4. 画面上のコンボボックスから学年を選択し、「スタート / つぎの問題」をクリックします。
5. テキストボックスに答えを入力し、「こたえあわせ」をクリックします。
